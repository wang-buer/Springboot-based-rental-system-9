# 基于springboot的租房系统9



## 1、项目介绍

基于springboot和vue租房系统9拥有三种角色

管理员：房东管理、房屋类型管理、房屋管理、预约管理、定金管理、租赁订单管理、评价管理等

租客：租房、预约房、付定金、评价房子等

房东：房屋管理、预约看房管理、定金管理、订单管理、评价管理等

## 2、项目技术

后端框架：springboot、Mybatis

前端技术：VUE、html

## 3、开发环境

- JAVA版本：JDK1.8（最佳）
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.7、8.x版本都可） 
- tomcat版本：无需
- maven版本：无限制
- 硬件环境：Windows
